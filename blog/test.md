# Input.pub

Input here, publish anywhere

## What?

一个支持 markdown 的输入框 + 一个按钮。帮助你把你撰写的内容轻松发表到各个地方。包括 GitHub, X, 邮件，ChatGPT, Gemini, Claude....

## Why?

目前写作时的痛点：

* 写一个简单的 x post, 打开 x 之后容易不自觉的刷了起来，其他社交平台也是如此。我想要一处不受干扰，让我可以专心把想法写下来的地方。

* 写博客的时候，我需要打开 vscode, 新建文件，写完之后推送到 GitHub。有点麻烦。

* 通常，一个想法，一篇文章写完之后我想要分享到不同的地方，比如同步到自己博客，程序员论坛，公众号，x, 等等。现在到处手动复制粘贴，略麻烦。希望把这个过程尽量简化

## How?

你可以直接编辑这段文字，然后点击右上角的 publish 按钮。把你的想法发送到各种平台。
